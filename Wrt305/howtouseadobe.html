Creating a WRT Logo in Adobe Illustrator

Materials Needed:

  • A Laptop / Desktop
  • Adobe Illustrator (7-day free trial available)
  • Computer Mouse (recommended but not necessary)

Instructions:

  1. Open Adobe Illustrator and create a new document.
  2. Set your workspace to "Essentials Classic" (Window > Workspace > Essentials Classic).
  3. Create the W:
    1. Use the Polygon Tool to create a triangle.
    2. Rotate the triangle 180Â°.
    3. Copy and paste the triangle.
    4. Position the triangles to form a W.
    5. Group the triangles.
  4. Create the R:
    1. Draw a rectangle.
    2. Create a circle and cut it in half.
    3. Adjust the half-circle to match the rectangle height.
    4. Use the Pen Tool to complete the R shape.
    5. Group the R components.
  5. Create the T:
    1. Copy the rectangle from the R.
    2. Add horizontal lines at the top using the Pen Tool.
    3. Group the T components.
  6. Arrange the letters to complete your WRT logo.
Note: To make shapes transparent, select the object, go to the color panel, and click the white box with a red line (no fill option).

For detailed troubleshooting, please refer to the original instructions or consult Adobe Illustrator's documentation.
